# Brand assets — drop your logo packet here

Put the whole packet in this folder (public/brand/). Helpful filenames:

- logo-full.svg        → header wordmark + mark (preferred: SVG)
- logo-mark.svg        → icon/mark only (square)
- logo-mark-light.svg  → mark for dark backgrounds (footer), if provided
- favicon.ico          → 32x32 (and/or 16/48) classic favicon
- icon.png             → 512x512 app icon (PNG, transparent or paper bg)
- apple-icon.png       → 180x180 for iOS home screen
- og.png               → 1200x630 social share image (optional)

Then tell me they're here and I'll:
1) swap the header's inline SVG for logo-full / logo-mark
2) set up favicon + app icons (Next conventions)
3) hook up the social-share (OG) image
