# çözümebak — Logo & Brand Package  (v2.0)

Complete visual-identity kit. All vectors are self-contained SVG (text outlined —
no font files needed to render).

## What's new in v2
Primary wordmark is now TWO-TONE: navy "çözüme" + coral "bak" — revealing the
Turkish phrase "çözüme bak" = "look at the solution." Solid-navy wordmark is kept
as the documented alternate for very small sizes and single-colour print.

## Typography
Quicksand — SemiBold for wordmark/display, Medium for body. Rounded geometric;
its terminals echo the round-capped graph line.

## Usage rules
- Clear space ≥ the badge corner radius on all sides.  Minimum digital icon: 32 px.
- On dark surfaces use the reversed / white variants.
- Never recolour, stretch, rotate, add effects, or alter proportions.
